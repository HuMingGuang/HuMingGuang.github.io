C++语言基础

C++文件结构

  • 头文件
    • 使用<>格式引用系统头文件，编译器会在系统库文件目录下搜索头文件
    • 使用””格式引用自定义头文件，编译器会现在当前工程目录下搜索头文件，再搜索系统库文件的目录

C++语言基本要素

  • 标识符：标识符的首字符必须是字母或下划线，而不能是数字

  • int min=10,max=min+50是合法的

  • double max=double() //初始化max为0.0

  • 变量的存储类型

    • extern类型

      • 变量的声明与定义：变量的声明是告诉编译器变量的名称和数据类型；变量的定义将为变量分配存储区域
      • 在一个文件中定义一个全局变量int var=0;在另一个文件通过使用extern声明全局变量extern int var;,在该文件即可访问全局变量var

    • static类型

      • 该变量分派在一个持久的存储区域，函数调用结束，并不会被释放，依然保留其值

    • register类型

      • 变量被放在cpu寄存器上，比普通的变量访问快

    • auto类型

      • 全局变量、静态变量属于静态存储

数据类型

  • 实数相加有效数字有限，float有效位是6～7，如果86041238.78与5相加，输出结果是86041245.00而不是86041243.78，个位四舍五入，小数位忽略。

  • 数组

    • 可以定义部分元素赋值：int array[10]={0,1,2,3}

    • 数组全部初始化为0:int array[10]={0}，但是不能初始化其他的值,int array[10]={1}//数组中第一个元素为1，其他元素是0

    • 如果对数组全部元素初始化，可以省略长度，int array[]={0,1,2,3,4,5,6,7}

    • 二维数组也支持对部分元素初始化

      结果是对每一行第一个元素赋值，其他元素为0

    • 如果提供全部元素的初始值，可以省略第一维的长度，但不能省略第二维int array[][2]={1,2,3,4,5}数组应该包含6个元素，最后一个元素默认初始化为0

  • 结构体

    • 定义结构体时，如果只需定义一次结构体变量（其他地方不需要定义该类型的结构体变量），可以不写结构体的名称，而只给出结构体变量

    • 结构体变量可以相互赋值

    • 结构体字节对齐问题

  • 共用体

    • 共用体中的成员共享同一个内存地址，其中一个成员改变，其他成员的值也会改变

  • 指针类型

    • 指向数组中第一个元素的指针

    • 指针常量：不能修改p指向的值，可以修改p指向的地址，指向的数据是常量

    • 常量指针：不能修改p指向的地址，可以修改p指向的变量的值，地址是常量

  • 引用类型

    • 引用相当于给目标对象起”别名”，修改引用对象就相当于修改目标对象

  • 自定义类型

    • 自定义类型并不是真的创建新的数据类型，而是为存在的数据类型定义一个新的名称

    • 能够提高程序的移植性

运算符

  • 算术运算符

  • 关系运算符

  • 自增自减运算符
  • 位运算

    • 使用异或交换两个变量的值(不使用中间变量)

    • 左移运算

    • 右移运算

  • sizeof运算符

    • 如果使用的是32为系统，sizeof运算符测试指针的长度时，无论指针的类型及指向什么类型的数据，指针长度都为4，实质是测量的是数组的首地址

  • new和delete运算符

    • 栈存储通常用于存储占用空间小、生命周期短的数据，如局部变量和函数参数；堆存储通常用于存储占用空间大、生命周期长的数据，如静态数据、全局数据。

    • 堆的初始化

表达式

  • 逗号表达式

  • 三目运算符

语句

  • do while

  • goto语句

  • return 之前一定要记得释放内存

  • exit语句：终止当前调用的进程，结束应用程序；return 则退出当前调用进程的函数。

面向过程程序设计

函数

  • 前置声明
  • 默认值参数

  • int (&array)[10]定义一个引用对象，指向10个元素的数组，在函数中引用时会检查元素个数

  • 设置指针/引用参数-函数传值方式

    • 值传递：在函数调用时，将实际参数的值复制一份传递到函数中，所以并不会影响实际参数的值

    • 引用传递：在函数调用时，会修改实际参数的值

    • 使用指针或引用作为函数参数，均采用引用方式传递；对于引用类型（&），引用必须被初始化为一个对象，并且不能使它再指向其他对象。

  • 重载函数

    • 定义重载函数

    • 函数的返回类型，不作为区分重载函数的一部分

    • const关键字不作为重载函数的一部分

  • 指针类型或引用类型，const可以作为重载的一部分

  • 使用typedef自定义类型不作为重载的标识

  • 局部域中声明的函数不是重载函数全局域中的函数

  • 函数指针-增加程序的灵活性

作用域和生命期

  • 全局变量，如果没有初始化，其值为0；但是对于局部变量，没有初始化，其值不可预期。

函数模板

  • 函数模板提供一种机制，使函数返回值、参数类型能够被参数化，而函数体保持不变。

  • 模板的重载

面向对象的程序设计

类和对象

  • 类成员的访问

    • public：公有成员，在程序的任何地方都可以访问；
    • protected：保护成员，只能在该类和该类的派生类（子类）中访问，不能被该类的对象访问；
    • private：私有成员，该成员只能在该类中访问。
    • 定义类对象：User user; 声明为指针类型：User *user;声明为指针类型，并分配内存空间User *user()=new User;或User *user=new User();

  • 构造函数和析构函数

    • 构造函数在定义对象的时候被调用，析构函数在对象超出作用范围或使用delete释放对象时被调用。如果用户没有提供，则系统默认给出。

    • 构造函数

      • 类体中定义数据成员时，为其直接赋值是非法的，如果需要给常量和引用类型数据成员初始化，则需要用:

    • 析构函数：析构函数没有参数，因此不能重载

    • 内联成员函数：对于内联成员函数，程序会在函数调用的地方直接插入函数代码，如果函数体语句较多，会导致程序代码膨胀

    • 静态类成员

      • 将类成员定义为静态类成员，则允许使用类名直接访问，使用static 标识

      • 是被所有的类对象共享的，即无论静态数据成员只有一份，如果某一个对象修改静态数据成员，其他对象的静态数据成员也将被修改

      • 注意一

      • 注意二

      • 类的静态成员函数只能访问类的静态数据成员

      • 静态成员函数不能在类体外定义

    • 运算符的重载

      • “+”运算符重载

      • 重载之后的运算符不具有交换性

    • 友元类和友元方法

      • 希望另一个类访问当前类的私有成员

      • 只允许另外类的某个成员访问当前类的私有成员

    • 类的继承
      • 类的继承的定义

    • Virtual关键字（虚方法）

      • 使用虚方法实现类的动态绑定，即根据对象运行时的类型来确定调用哪个类的方法，而不是根据对象定义时的类型来确定调用哪个类的方法；

    • 纯虚方法/抽象方法

      • 一个包含纯虚方法的类称为抽象类，抽象类是不能被实例化的，通常用于实现接口定义，是在虚方法的 定义上末尾加”=0”。

      • 抽象类通常作为其他类的父类，从抽象类派生的子类如果也是抽象类，则子类必须实现父类的所有纯虚方法

      • 子类对象的创建和释放过程

      • 定义一个基类型的指针，调用子类的构造函数为其构建对象。释放时，如果析构函数是虚函数，则先调用子类的析构函数，在调用父类的析构函数；如果析构函数不是虚函数，则只调用父类的析构函数，存在内存泄漏问题。所以，一般写类的析构函数时，通常是虚函数。

    • 多继承

      • 允许子类从多个父类继承公有和受保护的成员

      • 虚继承

        当Car和Bird都继承于Thing这个基类时候，如果Airplane继承了Car和Bird,则需要在Car和Bird把Thing作为虚继承（即class Bird:virtual public Thing{ …}），这样Airplane才不会创建两个Thing类的备份。

    • 类域：自定义类型应该放在类成员定义的前方

    • 局部类：可以在函数内部定义类，但是其作用范围不能超过该函数范围
