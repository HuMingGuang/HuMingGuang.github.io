一、维度灾难

在高维情形下出现的数据样本稀疏、距离计算困难等问题是所有机器学习方法共同面临的严重障碍，称作“维度灾难(curse of dimensionality)。

缓解维度灾难的一个重要途径是降维(dimension reduction)。

降维之所以有效的原因是：人们观测或者收集到的数据样本虽然是高维的，但是与学习任务密切相关的也许仅仅是某个低维分布，即高维空间中的一个低维“嵌入”。

常见的降维方法

监督降维算法：线性判别分析(Linear Discriminant Analysis:LDA) 。

无监督降维算法：主成分分析PCA(Principal Component Analysis) 。

  3. 相比于PCA是对样本的协方差矩阵$\sum$求解特征矩阵和特征向量作为投影向量，LDA使用的是$S{w}^{-1}S{B}$($S{w}^{-1}$:类内散度矩阵,$S{B}$)的求解特征矩阵和特征向量作为投影向量。

四、局部线性嵌入LLE

  1. 相比于PCA和LDA等关注样本方差的降维方法，LLE(Locally Linear Embedding)更关注于降维时保持样本局部的线性特征。例如：

    经过LLE降维后，希望$x1$在低维空间对应的投影$$x{1}^{‘} x{2}^{‘} x{3}^{‘}x_{4}^{‘}$$仍然保持线性关系

  2. LLE 算法步骤

    第一步：计算样本$xi$的$K$个最近邻$x{i1},x{i2},\cdots,x{iK}$

    第二步：求出局部协方差矩阵$Z_{i}=(x_i-x_j)(x_i-x_j)^{T}$，并计算相应的权重系数向量

    第三步：权重系数向量$W_i$组成权重系数矩阵$W$，计算$M=(I-W)^{T}(I-W)$

    第四步：计算矩阵$M$的中最小的$d$个特征值对应的特征向量$v_1,v_2,\cdots,v_d$，一般第一个最小的特征值值为0舍弃.这$d$个特征向量组成的特征矩阵即为降维后的样本集

  3. 小结

    • 优点
      • 学习任意维的局部线性的低维流形（局部具有欧式空间性质）
      • 稀疏矩阵特征分解，计算复杂度较小的
    • 缺点
      • 流形只能是不闭合的，且样本集是稠密均匀的
      • 对最近邻样本的选择敏感

五、 PCA、SVD、LDA的区别与联系

名称 区别与联系
PCA PCA是属于一种线性、无监督、全局的降维算法。目标：选择方差大的投影面。应用：PCA在语音识别中，可以过滤音频中固定的频率的背景噪声；人脸识别的脸部特征提出中。
SVD 可以用于PCA 和 LDA中，避免做特征分解（开销大）。直接根据SVD求出右奇异矩阵，而右奇异矩阵可以用于列数（即特征维度）的压缩。
LDA LDA是有监督的降维算法，相比于PCA可以对有类别信息的数据进行降维处理。目标：选择在投影后类内方差小，类间方差大的投影面（也就是将数据投影，使同一类别的数据投影尽可能接近，不同类别的距离尽可能大）。应用：在语音识别中，可以对每个人的语音信号进行区分（因为LDA可以对有类别信息，做降维处理）。LDA除了可以做降维，还可以做分类。
