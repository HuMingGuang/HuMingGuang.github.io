一、Python的类和对象

1.一个class定义产生一个用户定义类型，可以向内置类型一样创建对象，称为该类的实例。

2.一个类里可以定义一个名字为__init__的初始化函数，在创建这个类的实例时，解释器将自动调用这个函数，建立对象的初始状态。

3.Python没有说明属性只限制内部访问的专门机制，只能通过编程来约定保护。一般约定：类定义里由一个下划线_开头的属性名（包括函数名）当作内部名，不应该在类外使用。如果在类里出现以两个下划线开头的属性名，解释器会把它们换名，使在类定义之外直接写属性名时无法找到该属性，这是一种保护。

5.定义类方法

定义类的方法时加@classmethod，用于实现与这个类本身相关的操作，其参数表里至少要有一个表示定义类的形参，习惯用cls类方法需要通过类名，以属性引用的形式调用，执行时这个类约束到方法的cls参数，可以通过cls访问其它属性。

二、继承

1.派生类：通过继承定义的新类；基类：被继承的类

2.通过继承定义派生类

3.派生类对象的构建

4.基于继承的平面图形类定义

5.动态约束：基于方法调用时self所指定实例对象的类型确定被调用函数；通过动态约束确定调用关系的函数，称为虚函数

从对象x调用方法m时，需要确定被调用函数，确定方式是从对象所属的类开始找，如果这里有，调用的函数就确定了；如果没有，解释器到该类的基类去继续找。

6.标准函数super:用于调用本类覆盖的的基类方法

7.循环移位表

继承的一种用途是扩充已有的类、增添新功能，或对已有功能做些改动。如果某个已有类型基本满足要求，但还缺少若干必要功能，或有少许功能不合乎需要，由已有类型派生一个满足需要的新类，就是一种合理而且经济的做法。

循环移位表：在list的所有功能的基础上，还支持两个循环移位操作。即能把表中最前的几个元素依次移到最后，或把最后几个元素依次移到最前面。

8.多继承

定义派生类时经常只指定一个基类，这种继承为单继承；如果指定多个基类，则为多继承。
