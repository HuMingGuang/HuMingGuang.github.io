1.从数据规模上进行分类

样本类别分布不均衡从数据规模上可以分为大数据分布不均衡和小数据分布不均衡。大数据分布不均衡，整体数据规模较大，只是其中小样本类的占比较少；小数据分布不均衡，整体数据规模较小，并且占据少量样本比例的分类数量也少，这种情况下无论如何也无法实现完整特征值的覆盖。

2. 哪些场景容易出现样本不均衡

异常检测场景、客户流失场景、罕见时间的分析、发生频率低的事件

3.解决方法

3.1 过采样和欠采样

过采样(over-sampling)：通过增加分类少数类样本的数量来实现样本均衡，最直接的方法是复制少数类样本，但是如果样本特征少，容易导致过拟合问题；改进的方法通过少数类加入随机噪声、干扰数据、规则，例如SMOTE方法。

欠采样(under-sampling)：通过减少分类中多数类样本的数量来实现样本均衡，最直接的方法是随机去掉一些多数类样本来减少多数类的规模，缺点是会丢弃一些重要信息。

3.2 正负样本的惩罚权重

对分类中不同样本数量的类别分别赋予不同的权重，一般思路小样本量类别权重高，大样本量类别权重低；有些模型算法支持自动均衡处理，例如：SVM。

3.3 组合/集成方法

组合/集成方法指的是在每次生成训练集时使用所有分类中的小样本量，同时从分类中的大样本量中随机抽取数据来与小样本量合并构成训练集，这样反复多次会得到很多训练集和训练模型。最后使用组合方法（投票/加权投票）产生分类预测结果。

3.4 特征选择方法

上述三种方法都是基于数据行的操作，通过多种途径可使不同类别的样本数据行记录均衡。除此之外，也可考虑给予列的特征选择方法。

一般情况下，样本不均衡也会导致特征部分不均衡，但如果小类别样本量具有一定的规模，那么意味着其特征值的分布较为均衡，可通过选择显著型的特征参与解决样本不均衡问题。
