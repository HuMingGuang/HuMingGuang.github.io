1.3 参数调整

模型参数分为两种

  • 树具体参数
    • min_samples_split
    • min_samples_leaf
    • max_depth
    • min_leaf_nodes
    • max_features
    • loss function
  • Boosting具体参数
    • n_estimators
    • learning_rate
    • subsample
**第一个问题：为更高的学习速率寻找估计器参数** **第二个问题：调整树的具体参数** **第三个问题：调整下采样数和更低的学习速率** 所有的调整都是为了降低学习速率和增加估计器数量，从而得到更高鲁棒性的结果

2. XgBoost训练调参过程

  • 确定学习速率和tree_based 参数调优的估计器数目
  • max_depth 和 min_weight 参数调优
  • gamma参数调优
  • 正则化参数调优
  • 降低学习速率

XgBoost算法是串行处理的，但是工程上xgboost还提出了一种可并行的近似直方图算法，对每个特征高效地生成候选的分割点。
