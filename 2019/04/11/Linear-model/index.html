3.2 最小二乘与极大似然的区别

  • 最小二乘法：即观测值与实际数据误差平方和最小，其没有假设，几何意义上就是距离最小；

  • 极大似然估计：估计参数可能值，使样本发生概率最大。

4. 最小二乘的局限性

  • 首先，最小二乘法需要计算$A^{T}A$的逆矩阵，有可能它的逆矩阵不存在，这样就没有办法直接用最小二乘法了，此时梯度下降法仍然可以使用。当然，我们可以通过对样本数据进行整理，去掉冗余特征。让$A^TA$的行列式不为0，然后继续使用最小二乘法。

  • 第二，当样本特征$m$非常的大的时候，计算$ A^{T}A$的逆矩阵是一个非常耗时的工作$(m+1) \times (m+1)$的矩阵求逆），甚至不可行。此时以梯度下降为代表的迭代法仍然可以使用。那这个$m$到底多大就不适合最小二乘法呢？如果你没有很多的分布式大数据计算资源，建议超过10000个特征就用迭代法吧。或者通过主成分分析降低特征的维度后再用最小二乘法。

  • 第三，如果拟合函数不是线性的，这时无法使用最小二乘法，需要通过一些技巧转化为线性才能使用，此时梯度下降仍然可以用。
  • 第四，讲一些特殊情况。当样本量$N$很少，小于特征数$m$的时候，这时拟合方程是欠定的，常用的优化方法都无法去拟合数据。当样本量$N$等于特征数$m$的时候，用方程组求解就可以了。当$N$大于$m$时，拟合方程是超定的，也就是我们常用于最小二乘法的场景了。
