1.5 Softmax 回归 vs k 个二元逻辑斯谛回归

  • 如果你在做分类的时候，类别之间存在互斥关系，也就是你训练的每个样本只能打上一个标签，则应该选择使用softmax 回归；
  • 但是如果类别之间没有互斥关系，每个训练样本都可以有多个标签，则应该选择使用k个二元逻辑斯谛回归，在训练第个分类器的时候，每个样本被判别是否属于第类。

1.6 逻辑斯谛回归的优缺点

优点

  • Logistic回归模型直接对分类的可能性进行建模，无须事先假设数据满足某种分布类型
  • Logistic回归模型不仅可以预测出样本类别，还可以得到预测为某类别的近似概率
  • Logistic回归模型中使用的对数损失函数是任意阶可导的凸函数，可以避免局部最小值问题
  • Logistic回归模型可以很好的处理稀疏高维特征，比如广告点击率预测问题

缺点

  • Logistic回归模型本质上是一种线性模型，只能做线性分类，不适合处理非线性的情况
  • Logistic回归模型对正负样本的分布比较敏感，所有要注意样本的平衡性

2. 岭回归及Lasso回归

岭回归与Lasso回归的出现是为了解决线性回归出现的过拟合以及在通过正规方程方法求解的不可逆这两类问题，这两种回归通过在损失函数中引入正则化项来解决这两个问题。Lasso是基于惩罚方法对样本数据进行变量选择，通过对原本的系数进行压缩，将原本很小的系数直接压缩至0，从而将这部分系数所对应的变量视为非显著性变量，将不显著的变量直接舍弃。
