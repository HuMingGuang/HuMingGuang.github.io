一、基本语义问题

1.变量和对象

序号 c语言 Python
1 值语义 引用语义
2 变量必须声明类型，以便安排存储，不同类型的变量大小不同，也导致一种类型的变量不能保存其他类型的值。 变量只需保存对象标识，所有变量大小相同。变量不需要说明，赋值即定义。变量没有类型，一个变量完全可以一会关联整数，一会关联字符串。
3 缺点：由于变量无类型，解释器不能根据代码中变量确定操作是否正常进行，必须在运行中检查。运行中频繁检查时Python程序效率低的一个重要原因。

2.赋值和共享

在Python中数据值是独立对象，赋给变量就是建立关联，以便通过变量使用对象。

具体情况参考：深拷贝与浅拷贝

3.函数实参和共享

函数调用时，形参先约束到实参表达式算出的对象，在函数里重新给某个形参赋值，则使它约束到另一个对象，这种操作不会影响函数调用的实参。
