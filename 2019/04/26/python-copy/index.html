  • 直接赋值：其实就是对象的引用（别名）。

  • 浅拷贝(copy)：拷贝父对象，不会拷贝对象的内部的子对象。

  • 深拷贝(deepcopy)： copy 模块的 deepcopy 方法，完全拷贝了父对象及其子对象。
  • 总结：赋值是原对象变它也变，浅拷贝是原对象的子对象变它才变，深拷贝是原对象的子对象变，它也不变

三、解析

3.1 b = a:赋值引用，a 和 b 都指向同一个对象

3.2 b = a.copy(): 浅拷贝, a 和 b 是一个独立的对象，但他们的子对象还是指向统一对象（是引用）

3.3 b = copy.deepcopy(a):深度拷贝, a 和 b 完全拷贝了父对象及其子对象，两者是完全独立的

五、总结

首先得搞清楚你的对象是可变对象，还是不可变对象。如果是不可变对象a，则在赋值之后就已经创建了新的对象b，如果此时改变对象a的值，但是b对象的值还是不变；如果是可变对象，则要根据上述第三章的情况讨论。
