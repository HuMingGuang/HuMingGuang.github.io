1. 集成学习基础知识

1.1 集成学习的种类

种类 训练方式 核心思想 典型算法
Bagging 各个弱分类器无强依赖，可以并行训练 多个弱分类器有放回地采样，单独学习。最终做决策的时候，进行集体投票，集体决策。各个弱分类器权重相同。采用分而治之的策略，通过训练样本多次采样，并分别训练出多个不同的模型，然后做综合，来减少弱分类器的方差。 Random Forest(Bagging with CARTs)
Boosting 弱分类器串行训练 将弱分类器层层叠加，每一层在训练的时候，对前一层弱分类器分错的样本，给予更高的权重，并在本层分类器训练后，计算分类误差，更新本层分类器的权重。测试时，根据各层分类器的结果加权得到最终的结果。通过聚焦弱分类器的分错的样本，减小弱分类器的偏差。 Adaboosting、GBDT

1.2 集成学习基本步骤

  • 找到误差相互独立的弱分类器
  • 训练弱分类器
  • 合并弱分类器的结果

合并分类器的方法有两种：

  • voting：用投票的方式，将获得最多选票的结果作为最终结果；
  • stacking：用串行的方式，把前一个弱分类器的结果输出到下一个分类器，将所有弱分类器的结果相加作为最终结果输出。

1.3 最常用的弱分类器：决策树

  • 决策树可以较为方便地将样本的权重整合到训练过程中，而不需要使用过采样的方式来调整权重；
  • 决策树的表达能力和泛化能力，可以通过树的结构来调节
  • 数据样本的扰动对决策树的影响较大，不同子样本集合生成的决策树弱分类器随机性较大。

4. 梯度提升决策树(GBDT)

4.1 核心思想

根据当前模型损失函数的负梯度信息来训练新加入的弱分类器，然后将训练好的弱分类器以累加的形式结合到现有模型中。

4.3 GBDT优点和局限性

优点

  • 可以灵活处理各种类型的数据，包括连续值和离散值。
  • 在相对少的调参时间情况下，预测的准确率也可以比较高。这个是相对SVM来说的。
  • 使用一些健壮的损失函数，对异常值的鲁棒性非常强。比如 Huber损失函数和Quantile损失函数。
  • 在分布稠密的数据集上，泛化能力和表达能力都很好

缺点

  • 在高维稀疏数据集上，表现不如支持向量机或者神经网络
  • 在处理文本分类特征上，相对于其他模型的优势不如在处理树枝特征时明显
  • 训练过程需要串行训练，只能在决策树内部采用一些局部并行的手段提高训练速度

5. XGBoost

GBDT与XGBoost的区别与联系

GBDT XGBoost
GBDT是机器学习算法 XGBoost是GBDT 算法的工程实现
XGBoost显式地加入正则化项来控制模型的复杂度，有利于防止过拟合，提高模型的泛化能力
GBDT在模型训练时只使用代价函数的一阶导数信息 XGBoost对代价函数进行二阶泰勒展开，可同时使用一阶和二阶导数
采用CART作为弱分类器 支持多种弱分类器
在每次迭代时使用全部数据 采用与随机森林相似的策略，支持对数据进行采样
无法对缺失值处理 自动学习出缺失值的处理策略
