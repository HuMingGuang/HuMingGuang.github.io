一、表和元组

1.表(list)

2.元组(tuple)

元组与list的最大区别是，创建之后不能修改，结构不能变（不能加入或者删除元素），其中元素不能变（不能给元素赋值），其他方式与list列表一致

二、序列和序列操作

1.序列不是python类型

序列不是python类型，而是涵盖具有共同性质的一些类型的概念，序列对象可以包含顺序排列的多个元素，通过下标访问元素，下标从0开始。标准类型str、list、tuple都是序列类型，但是它们之间存在差异，主要是在对象能保存的元素（表和元组可以是任何对象，字符串的元素只能是字符），对象创建后能否修改（list的对象可以修改，tuple和str的对象不能修改）。

6.表操作

标准函数sorted可用于任意迭代对象，结果是表。但是sorted(list1)得到表list1的排序拷贝（是一个新表），list1不变；list1.sort()则直接调整list1表中的元素，使之按顺序重新排列。

标准序列反转函数reversed可用于各种序列对象，返回迭代器（不是序列，要转化为序列需要加list(reversed(list1))），而本身的list1并没有变；但是list1.reversed()直接在list1上操作。

7.描述式

使用执行语句的方式构造复杂的序列

8.bytes和bytearray类型

字节（bytes）就是8位的二进制串，元素范围从0到255表示字符的编码，b1=b'this is a byte'，必须加前缀b或B，支持不变序列操作

字节数组（bytesarray）：支持可变序列操作

9.array类型

array一种专门用于表示数值序列的类型，存储比较紧凑，操作效率比表高，是一种可变类型。

如果计算中需要很大的数值向量或矩阵，用array既节约存储也节省时间。如果数值序列不太大，直接使用列表更方便。

五、字典

1.字典的类型名

字典的类型名是dict，其定义域是任意的关键码（键）的集合，值域是任意对象结合。允许把任意的键及其关联值存入字典，可以通过键获取与之关联的值。字典的键必须是不变对象。

dic1={'math':112,'phys':212}

字典操作：

dic[k]=v：在dic里记录k的关联值为v

dic[k]：得到dic里与k关联的值

2.使用类型名dict从二元组的表构造字典

faculty = dict{[('math',112),('phys',211)]}

3.字典的键都是字符串

faculty = dict(math=112,phys=211)

4.描述式生成字典

n: n**3 for n in range(4) #得到 {0:0,1:1,2:8,3:27}

六、集合

1.集合类型

集合类型分为set和frozenset；分别是可变类型和不可变类型

2.集合具有唯一性

集合具有唯一性，不会出现重复元素，元素之间没有顺序关系。

3.集合与字典形式很相似

集合与字典形式很相似，只不过集合不存在键-值对，集合中元素描述一般是表达式，或者一个个字符/字符串

4.空集合

5.标准库包pickle

pickle包可以方便地把系统类型的对象介入文件，又能非常方便地重新装入，恢复以前保存的对象，其支持数据持久化。

pickle用二进制文件存储数据，以避免输入输出中数据转换的计算开销和可能误差，也使存储更密集，少占用外存空间。
